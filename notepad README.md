# Web Vulnerability Scanner

A Python-based CLI tool to detect basic web vulnerabilities like Cross-Site Scripting (XSS) and SQL Injection.

## Features
- Form detection and analysis
- Multi-payload XSS testing
- SQL Injection detection (error-based)
- JSON report generation
- Timestamped scan reports

## Tech Stack
- Python
- requests
- BeautifulSoup

## Usage

```bash
python scanner.py https://example.com
